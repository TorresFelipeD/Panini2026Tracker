# Release Notes

## 0.1.0 - 2026-04-20

- Se crea la primera versión local de “Álbum Mundial 2026 Tracker”.
- Se implementa backend .NET 8 Web API con capas `Api`, `Application`, `Domain` e `Infrastructure`.
- Se incorpora SQLite con migración inicial, seed desde JSON y almacenamiento de estado del usuario, repetidas, imágenes y logs.
- Se construye frontend Angular standalone con módulos de álbum, repetidas, imágenes y logs.
- Se añade modo oscuro, versionado centralizado, SVGs base, documentación inicial y licencia MIT.
